⚙️ Supabase Setup Required
Configure your Supabase project to start using the system.
First-time setup: Replace YOUR_WORKER_URL with your Cloudflare Worker URL at the top of this file's JavaScript section.

Step 1: Create Supabase project at supabase.com and run SQL schema below

Step 2: Create Cloudflare Worker at dash.cloudflare.com → Workers & Pages → Create

Step 3: Paste the Worker script (gpm_worker.js) and add secrets: SUPABASE_URL, SUPABASE_ANON_KEY, CASHIER_PASS

Step 4: Copy your Worker URL (e.g. gpm-fee.yourname.workers.dev)

Step 5: Replace YOUR_WORKER_URL in this HTML file with your Worker URL
